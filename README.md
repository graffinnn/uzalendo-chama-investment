# Uzalendo Chama Investment Group — Mobile Application

A mobile application for managing member contributions, rotating payout cycles, emergency loans, voluntary savings, and investment portfolios for the Uzalendo Chama savings group in Kenya.

## Technology Stack

| Layer | Technology |
| Mobile Frontend | React Native + Expo |
| Backend API | Node.js + Apollo Server (GraphQL) |
| Database | MySQL |
| Authentication | JWT |
| Version Control | Git / GitHub |

## Core Modules
- User Management (Member & Admin roles)
- Contribution Tracking (KES)
- Cycle Management (rotating payout, random assignment)
- Savings with interest distribution
- Emergency Loan Management (score ≥ 50 required)
- Member Reliability Scoring
- Investment Portfolio Tracking
- Admin Dashboard
- Offline support + Push notifications
